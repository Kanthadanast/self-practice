
โจทย์ : นับจำนวนครั้งที่แต่ละคำปรากฏในประโยค

โดยที่ฉันตรวจสอบก่อนว่า (sentence) ที่รับเข้ามาเป็น null, undefined หรือ ('') รึเปล่า ถ้าใช่จะ return undefined เลย
ต่อมาฉันทำให้ข้อความทั้งหมดเป็นตัวพิมพ์เล็กโดยใช้ toLowerCase แยกคำออกจากกันด้วย split(' ')
ต่อมาฉันสร้าง keypairfreqword เอาไว้สำหรับเก็บผลลัพธ์
ต่อมาฉันใช้ for loop ใน array ถ้าเจอคำนี้ if (keypairfreqword[word]) ให้บวกจำนวนเพิ่มอีก 1 ถ้าเกิดไม่เจอให้กำหนดค่าเริ่มต้นเป็น 1
และสุดท้ายฉันมาเก็บไว้ที่ keypairfreqword และ return ออกมา
