โจทย์ : Write a function that takes a sentence (string) and returns the longest word in the sentence.

ฉันได้รับโจทย์มาว่าให้เขียนโปรแกรมที่หาคำที่ยาวที่สุดในประโยค
โดยเริ่มแรกฉันสร้าง function longestWord(sentence) ขึ้นมา ข้างในบรรทัดแรกฉันเขียนให้ตัวแปรหรือ result เป็น ''ช่องว่างไว้ 
ต่อมามาใช้ if เพื่อหาคำ โดย เช็คก่อนอันดับแรกว่า sentence เป็น string รึเปล่า
ต่อมาข้างในฉันให้ตัวแปร word เอา '' ที่เป็นช่องว่างระหว่างคำในประโยคออกไป
ต่อมาฉันใช้ for loop เพื่อหาความยาวตัวอักษร โดยที่ความยาวตัวอักษรต้องน้อยกว่า result ที่เก็บค่ามา
และสุดท้ายฉัน return result ออกมา
