โจทย์ : สร้าง object ที่มี key:value หลายแบบ

ฉันได้สร้าง object ที่มีชื่อว่า any และมี key : value ทั้งหมด 7 ตัว 
โดยที่ อันที่ 1 ฉันให้เป็น string ที่มีชื่อว่า 'sunday' อันที่ 2 เป็น number อันที่ 3 เป็น boolean ฉันเลือกเป็น false  
อันที่ 4 เป็น null อันที่ 5 เป็น array ที่เก็บค่าทั้งหมด 4 ตัว อันที่ 6 เป็น เลขทศนิยม 
และสุดท้ายอันที่ 7 เป็น key กับ value และมี value ข้างในอีกที โดยฉันให้ชื่อว่า pets และ  value มี name กับ function ที่จะ return ค่าเป็น "meow"

สิ่งที่ฉันได้เรียนรู้ :
1.key สามารถซ้ำได้หลายอัน แต่ value จะได้แค่ตัวสุดท้าย (นอกเหนือจาก code มาจากในคลาส)
2.สามารถมี value ซ้อนใน value อีกอันได้ รวมถึงมี function ในนั้นอีกด้วย
